## 升级

```sh
# 查看当前安装的Ruby源地址
$ gem source -l
https://ruby.taobao.org/

# 移除旧的Ruby源地址
$ gem sources --remove https://ruby.taobao.org/

# 添加新的Ruby源地址
$ gem sources --add https://gems.ruby-china.com

# 安装Cocopods
$ sudo gem install cocoapods

# 更新到最新版本
$ sudo gem install -n /usr/local/bin cocoapods --pre
```
